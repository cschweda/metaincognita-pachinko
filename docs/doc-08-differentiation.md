## 1. Competitive Landscape

### 1.1 Browser-Based Pachinko Simulations

| Product | Type | Physics | Education | Theming | Notes |
|---|---|---|---|---|---|
| **Phaser Pachinko Example** | Demo | Matter.js (basic) | None | None | Bare-bones pin grid + bucket. Proof of concept, not a game. |
| **Plinko clones** (various) | Games | Matter.js or Box2D | None | Minimal | Plinko (Price is Right) style — balls drop through pegs into multiplier buckets. No lottery, no gates, no modes. Not pachinko. |

| **Vintage Pachinko Simulator** | None found | — | — | — | No comprehensive browser-based simulator of modern Digipachi mechanics exists in the English-speaking web. |

### 1.2 Key Finding

**No English-language browser-based simulator faithfully reproduces modern Digipachi pachinko with real physics, a digital lottery, the kakuhen/jitan mode system, and educational statistical analysis.** This is an unoccupied niche.

Plinko clones are everywhere but are not pachinko — they lack the lottery, the mode system, and the cultural context. Crazy Pachinko is a real-money gambling product, not an educational tool. The Yakuza mini-games are fun but simplified. Pachislot apps are pure slots.

## 2. Differentiation

### 2.1 What Makes PachinkoParlor Unique

| Differentiator | Description |
|---|---|
| **Real physics + real lottery** | The only browser simulator combining Matter.js ball physics with an authentic Digipachi RNG lottery system. Balls actually cascade through pins; the lottery actually runs at configurable probability tiers. |
| **Full mode system** | Kakuhen (fever), jitan (fast cycle), koatari — the complete state machine that gives pachinko its dramatic arc. No other browser sim implements this. |
| **Educational statistics** | Real-time probability convergence charts, RNG transparency mode, house edge calculation, burn rate analysis. The tool exposes what pachinko is designed to hide. |
| **Themeable** | Five original visual/audio themes with a manifest-driven architecture that allows new themes to be added without engine changes. |
| **Cultural context** | Historical essay covering pachinko's origins, the three-shop system, zainichi Korean entrepreneurship, yakuza connections, and problem gambling — contextualizing the game within Japanese society. |
| **Suite integration** | Part of the Metaincognita casino simulator suite with consistent dark-mode UI, shared design system, and cross-simulator navigation. |
| **No real money** | Purely educational. No gambling, no microtransactions, no token purchases. |

### 2.2 Competitive Moat

The combination of **authentic physics + authentic lottery + educational analytics + cultural context** is the moat. Any one of these alone is achievable; the combination is what no existing product offers. The closest comparison would be the Wizard of Odds site's game analysis pages — but those are static math, not interactive simulations.
